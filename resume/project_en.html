GlobalConnect
GlobalConnect is a community platform for international students, providing various features to help them share information and network. This project was developed based on React and Node.js, with main features including bulletin boards, real-time chat, and event management. We used MongoDB for data management and implemented real-time communication through Socket.io. Additionally, we support stable hosting and deployment using AWS. GlobalConnect aims to make the lives of international students more convenient and enriching, and was completed through teamwork and technical challenges.

Database Performance Issue
Problem Identification
As the bulletin board feature became active, the number of posts and comments increased rapidly, causing database performance degradation. This led to longer loading times for posts and delayed server response times.
Problem Solution
We improved query speed by adding indexes to frequently accessed fields (such as creation date and author). We also optimized queries to minimize unnecessary data retrieval and modified them to selectively fetch only the required data. Through this process, we gained an understanding of the importance of database optimization and performance tuning. Additionally, we learned index design and sharding techniques for effective management of large-scale data. This experience has equipped us with the ability to solve database-related issues more quickly and efficiently in the future.
Real-time Chat Stability Issue
Problem Identification
After introducing the real-time chat feature, as the number of users increased, we encountered issues with delayed chat messages and frequent connection drops. This significantly degraded the user experience.
Problem Solution
We analyzed server logs and network traffic to identify the causes of Socket.io connection drops and message delays. We then scaled the Socket.io servers, used a load balancer to distribute the load, and periodically cleaned up unnecessary data and connections to prevent memory leaks.
